Utilizando Seletores Personalizados!

Utilizando id

id é um atributo que permite que você personalize sua tag. Na página HTML será colocado o id dentro da tag que você deseja personalizar juntamente com o nome dado a ela.
Exemplo: <h1 id="nome">

É importante lembrar que: no CSS o id se transforma em #

Utilizando class

Porém, o id é um atributo de uso ÚNICO, o que quer dizer que você não pode utilizar a mesma personalização para outras linhas utilizando a mesma id. Para isso existe o atributo class, que funciona da mesma forma que a id, porém permite que você aplique a mesma personalização em mais de uma tag.
Também é possível utilizar mais de uma class por tag, separando-as por um espaço: <h1 class="nome1 nome2"<h1>

É importante lembrar que: no CSS o class se transforma em .

Utilizando <span>

O span é utilizado para que você tenha a liberdade de utilizar uma class em qualquer ponto do seu texto, sem aplicar o estilo a toda a tag. Foi isso que fiz no texto acima, que está destacado em negrito e laranja.
