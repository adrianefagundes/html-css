Estilos Externos

Para utilizar estilos externos, basta utilizar a tag <link:css>
Com ele você poderá criar uma folha específica para seu código de estilo, diminuindo a quantidade de linhas no seu código de conteúdo.
Essa página foi estilizada com essa forma de utlizar os estilos!

O melhor de tudo é que dessa forma, se o seu site requer a criação de diversas páginas, você pode utilizar um único código de estilo para todas elas! Isso facilita muito o seu trabalho.
