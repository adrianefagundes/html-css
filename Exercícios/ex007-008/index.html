Lembrando dos tempos de Tumblr...

Negrito:

Para destacar um termo, usa-se a tag B (não semântica).

Mas também temos o mesmo destaque utilizando a tag semântica STRONG.

Itálico:

Igualmente, para dar ênfase em termos utiliza-se a tag I.

Ou então, podemos colocar essa mesma formatação a partir da tag (agora semântica) EM.

Marcatexto:

Para marcar um texto, utiliza-se a tag MARK.

Juntamente com CSS, podemos modificar a cor de fundo do marcador

Tamanho de texto pequeno:

Para diminuir um pouco seu texto, usa-se a tag SMALL.

Texto deletado:

Há a opção de marcar um texto como excluido para indicar que não deve ser considerado. Para isso usa-se a tag DEL.

Texto inserido

Se for necessário chamar atenção para um pedaço de texto que foi adicionado mais recentement e, pode-se usar a tag

Sobrescrito:

Para escrever coisas como x20+3, usa-se a tag SUP.

Subescrito:

Para escrever algo como H2O, usa-se a tag SUB.
