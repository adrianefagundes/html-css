O no wrap permite que os elementos não extrapolem o tamanho máximo estabelecido no container. Porém, quando adicionados muitos elementos, estes podem perder suas características para se ajustas no tamanho permitido.

Obs: Se os elementos tiverem estabelecidos um tamanho mínimo, haverá o "vazamento" do conteúdo, como no exemplo abaixo.
