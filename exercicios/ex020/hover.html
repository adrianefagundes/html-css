Utilizando hover

Nessa página estou utilizando o seletor hover. Esse seletor está relacionado ao estado de um determinado elemento. Se esse elemento está vazio, marcado, habilitado, etc.

PASSE O CURSOR DO MOUSE AQUI PARA VÊ-LO EM AÇÃO.

É possível esconder um texto com esse seletor, colocando um parágrafo DENTRO da div e utilizar o comando div > p { display: no CSS.
Com o display ao invés de esconder, podemos fazer o texto aparecer quando o cursor do mouse estiver em cima do texto hover.
